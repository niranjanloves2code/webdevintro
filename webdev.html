Introduction to Websites and its Functionalities

  1. Each and every computer has a browser.
  2. Websites must be displayed if we typed address on the searchbar.
  3. Each websites are connected with high capacity server and worlwideweb.
  4. The specified pages would have been already developed and saved in the server.
  1. Information Exchange would happen in request response basis between client and the server.
  2. Browser will act as client which sends request and Server sends back response.
  3. Browser helps us to display the information in the readable format.
  4. HTTPS protocols are established to serve this purpose.

What is HTTPS ?

HTTPS Stands for Hyper Text Transfer Protocol

All type of media files need to be traversed to the user via browser in the human readable format . So,HTTPS protocols are established to serve this purpose

What is HTTPS Request and Response ?

Browser as a Client , Request passes to server ,server sends back to browser where u will have the requested data in the human readable format .
